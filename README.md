
The Hotel Management System is a Java-based console application developed using object-oriented programming concepts. This system simulates core hotel operations such as managing guests, rooms, bookings, and checkout processes.

It demonstrates the implementation of Java Collections, OOP principles, and layered architecture (Entity → Service → Main).

Features
 Add and manage Guests
 Add and manage Rooms (Standard / Deluxe)
 Book rooms for specific number of nights
 Automatic total amount calculation
 Booking summary display

Technologies Used

Java (JDK 21 / Java SE)

Eclipse IDE

Object-Oriented Programming (OOP)

Java Collections (ArrayList)

How It Works

Guests and Rooms are created and stored using ArrayList.

The HotelService class handles:

Booking a room

Calculating total amount (Room Price × Nights)

Generating booking ID

After booking:

Booking details are displayed

Checkout process completes successfully

How to Run

Open the project in Eclipse.

Ensure JDK is properly configured.

Run Main.java.

View output in Console.
